# Block Explorers

* Block explorers for let you view, verify, and analyze onchain activity across transactions, addresses, contracts, and blocks on our EVM‑compatible zk L2.
* Because Payy is privacy‑first, some data will be selectively disclosed or privacy‑preserving by default. You may see obfuscated fields when interactions occur with the [PrivacyBridge](../protocol/privacybridge.md).
