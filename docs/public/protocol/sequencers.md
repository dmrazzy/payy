# Sequencers

Sequencers are responsible for ensuring the sequencing and validity of transactions on Payy. Sequencer operators will be rewarded with the PAYY token for their work in sequencing and producing blocks.

Due to the high performance requirement of Payy:&#x20;

* multiple sequencers will operate in parallel to sequence blocks
* multiple blocks will be produced per block allocation by a single sequencer

Selection of sequencers will be determined using Fast Hotstuff consensus protocol.
