# Privacy Layer

The Payy Privacy Layer is a high performance privacy rollup that enables native privacy pools for all ERC-20 tokens.

{% hint style="info" %}
The Privacy Layer inherits much of its design from Payy Network V1, as described in our original 2024 whitepaper - [L2 Ethereum ZK Rollup for Private and Compliant Transactions](https://polybase.github.io/zk-rollup/whitepaper.pdf)
{% endhint %}
